# CompuSmart Technologies Ltd.

### Africa's Premier Smart Technology Platform

CompuSmart Technologies is a modular, API-first hybrid platform designed to transition African institutions—enterprises, schools, clinics, and government parastatals—into the Intelligence Age. This platform bridges the gap between complex AI architecture and real-world results through a "Human-First" UX approach.

---

## 🚀 Key Features

### 1. Persona-Based Ecosystem
The platform is built as a centralized "Intelligence Hub" that routes users into six specialized, high-fidelity portals:
- **🏢 Business/Enterprise**: Smart digital assistants & bookkeeping automation.
- **🎓 Academy**: AI-enhanced LMS & tech skill development.
- **🏥 Health**: Digital patient records & secure clinical tools.
- **🤝 Talent**: Skill-verified recruitment & HR automation.
- **🏛️ Government**: B2G procurement & digital citizen support.
- **🎨 Creative/Media**: AI-driven content performance & case-study portfolios.

### 2. High-Performance Design
- **Immersive UX**: Real-time backgrounds and staggered motion sequences.
- **Bento Core**: A globally standard asymmetric organization of ecosystem services.
- **Adaptive Architecture**: A scalable, decoupled React frontend ready for microservice integration.

### 3. Localization
- **Multi-Lingual Support**: Citizen support in Arabic, Hausa, Yoruba, Igbo, and English.
- **Contextual Tools**: Load-shedding mitigation trackers and regional compliance auditing.

---

## 🛠️ Tech Stack

- **Frontend**: React.js 18 with TypeScript
- **Styling**: Vanilla CSS (Premium Glassmorphism Design System)
- **Motion**: Framer Motion (Framer-Motion/Motion)
- **Icons**: Lucide React
- **Routing**: React Router DOM v6

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mfoniso1/compusmarttech.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Project Structure
- `/src/pages`: Specialized portal implementations.
- `/src/components`: Shared UI components (Navbar, Footer, Layouts).
- `/src/layouts`: Responsive page templates.
